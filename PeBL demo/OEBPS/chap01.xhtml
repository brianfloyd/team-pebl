Setting up a PeBL book

Step 1: Get the PeBL extensions

Go to PeBL’s github repository to download the extensions. For each extension, you will need both the javascript file and the style sheet. Some, like the Content Morning extension, also have images. You will need to save those as well.

Step 2: Unzip an .epub

PeBL books are EPUB 3 files, which means that at their core they are HTML 5 files contained in a .zip file. In order to optimize a .epub for PeBL, you will need to unzip the epub. This process is different for PC and Mac users.

Please select your operating system using the selctor on the right to view specific directions.

Windows

In Windows, open the folder that contains the .epub. Then select the .epub and rename the file from file_name.epub to file_name.zip. Make sure you have the “File name extensions” box checked so you can change the file extension.

Then to unzip the epub, right click and choose “Extract all.”

Mac

Step 3: Optimize your book

When you first unzip an epub, you should see at least three elements in the folder: a META-INF folder, an OEBPS folder, and a mimetype file. The exact content and structure of the book will depend greatly on how the .epub was created and the software used to package it. However, the main folder you will work with is the OEBPS folder.
